# Memo App

ICS MEDIAの記事『若手フロントエンドエンジニアのためのバックエンド入門（後編）』のメモアプリのデモです。

このコードは、ホスティングしてデモアプリとして稼働させることを目的としています。

以下のURLから確認できます。

https://two51009-memoapp-demo.onrender.com/

## ディレクトリ構成

- packages
  - backend: バックエンド
  - frontend: アプリ本体

## 各コマンド

プロジェクトルートで以下を実行するとdevが立ち上がります。

```bash
npm run dev
```
